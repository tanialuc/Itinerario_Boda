# Itinerario de Boda - Tania y Manuel

Página web interactiva con el itinerario detallado de la boda de Tania y Manuel.

## Características

- Diseño responsivo que se adapta a todos los dispositivos
- Contador regresivo en tiempo real hasta la fecha de la boda
- Línea de tiempo visual de los eventos del día
- Detalles específicos de cada actividad
- Información importante para los invitados
- Misma paleta de colores que la invitación original
- Optimizado para rendimiento

## Estructura del Proyecto
itinerario-boda/
├── index.html # Página principal del itinerario
├── style.css # Estilos CSS
├── script.js # Lógica JavaScript y generación de QR
├── flores-izquierda.png # Imagen de fondo
└── README.md # Documentación

## Cómo usar

1. Clonar el repositorio
2. Asegurarse de tener el archivo `flores-izquierda.png` en la raíz
3. Abrir `index.html` en un navegador
4. El código QR se generará automáticamente

## Tecnologías utilizadas

- HTML5
- CSS3 (con variables CSS y Flexbox/Grid)
- JavaScript vanilla
- Librería QRCode.js (CDN)
